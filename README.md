## Sobre O Projeto

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

**Mixology Management** é um sistema desenvolvido para otimizar a **gestão de bares de eventos**, com foco no controle preciso de estoque. O projeto foi criado para resolver a complexidade da logistica de uma empresa de eventos, oferecendo funcionalidades para **cálculo automático de fichas técnicas** e gestão de compras.

### Instalação e Execução

#### 1. Backend (API)

Acesse a pasta do backend, crie um ambiente virtual para isolar as dependências e instale os pacotes necessários.

1. Navegue até a pasta do servidor:
   ```sh
   cd backend
   ```
2. Crie e ative o ambiente virtual:
* Windows
   ```sh
   python -m venv venv

   .\venv\Scripts\activate
   ```
3. Instale as dependências:
    ```sh
    pip install -r requirements.txt
   ```
4. Configure as variáveis de ambiente (similar ao modelo .env.example)

5. Inicie o servidor
    ```sh
    uvicorn main:app --reload
   ```
#### 2. Frontend

1. Navegue ate a pasta de interfaces
   ```sh
    cd frontend
   ```
2. Instale os pacotes
   ```sh
    npm i
   ```
3. Inicie o projeto
   ```sh
    npm run dev
   ```
